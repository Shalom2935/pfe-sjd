# References techniques

Ce dossier contient uniquement les sources techniques necessaires a la
tracabilite des hypotheses de modelisation.

- `rcc/` : fiche technique DIAM4100 utilisee pour le choix du tap, des niveaux
  de brillance et des caracteristiques du regulateur.
- `transformers/` : reference TI L-830 pour le modele de transformateur
  d'isolement.
- `cables/` : reference du cable utilisee pour les parametres de la ligne PI.
- `lighting/` : reference du feu LED utilisee pour les charges secondaires.
- `standards/` : norme FAA utile a la validation des transformateurs
  d'isolement.
- `articles/` : article scientifique de support pour la modelisation AGL.

Le depot etant prive, les PDF sont conserves pour audit interne. Pour une
publication publique, remplacer ces fichiers par des citations et liens
bibliographiques.
